# Flowcharts for Basic Python Programs

## If-Else
- Start → Input number → Check >0 → Print Positive / Check <0 → Print Negative / Else → Print Zero → End

## Loops
- For Loop: Start → Initialize i → i <= n? → Print i → i++ → Repeat → End
- While Loop: Start → Initialize count → count <= n? → Print count → count++ → Repeat → End
